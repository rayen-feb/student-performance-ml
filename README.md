## Project Overview

This project predicts **student performance** using machine learning.  
It focuses on predicting the **final grade (G3)** as a regression task and classifying students as **pass/fail** as a classification task.  

### Goals:
- Predict final grades using easy-to-measure variables
- Classify students as pass/fail
- Explore feature importance
- Visualize predictions and metrics
- Apply advanced ML models (Random Forest, Gradient Boosting, XGBoost, MLP)

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/rayen-feb/student-performance-ml.git
cd student-performance-ml
```
2.Install dependencies:
```bash
pip install -r requirements.txt
```
3.usage 
```bash
python src/models/regression.py
```
4.Run Classification Models
```bash
python src/models/classification.py
```
5.Generate Plots and Reports:
```bash
python src/visualization/reporting.py
```
